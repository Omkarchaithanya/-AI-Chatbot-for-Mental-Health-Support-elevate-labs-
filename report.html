🧠 MindEase PRO

AI-Powered Mental Health Support Chatbot — Project Report
Submitted by: Omkar Chaithanya
Platform: Elevate Labs
Date: March 2026

Abstract

MindEase PRO is a full-stack AI chatbot designed to provide accessible, empathetic mental health support. It combines real-time emotion detection, Retrieval-Augmented Generation (RAG) with a Cognitive Behavioural Therapy (CBT) knowledge base, voice input/output, mood tracking dashboards, and guided mindfulness exercises — all served through a responsive single-page web application. The system detects seven emotions (joy, sadness, fear, anger, disgust, surprise, neutral), applies a three-level crisis safety filter, and adapts its tone based on user preferences. Text-to-Speech supports English, Hindi, Kannada, Tamil, and Telugu.

Introduction

Mental health care faces a global accessibility gap — long wait times, cost barriers, and stigma prevent millions from seeking help. Conversational AI offers a low-barrier first point of contact, available 24/7. MindEase PRO addresses this by delivering a clinically-informed chatbot that is both technically robust and emotionally aware.

The project integrates transformer-based NLP models with Flask's real-time WebSocket infrastructure, a SQLite persistence layer, and a mobile-friendly dark-mode UI. It is packaged for local use, Docker deployment, and cloud hosting (Render.com), with a public repository on GitHub.

⚠️ Disclaimer: MindEase PRO is a supportive tool, not a substitute for professional mental health care. Crisis messages automatically surface 988 / 911 emergency resources.

Steps Involved in Building

  1. Requirements Analysis — Identified core features: emotion-aware chat, mood tracking, exercises, crisis handling, and TTS.
  2. Architecture Design — Planned Flask REST + WebSocket backend, SPA frontend, SQLite DB schema with Users, MoodEntries, ChatMessages, and ExerciseLogs.
  3. AI Model Integration — Loaded DistilRoBERTa for emotion classification; BlenderBot-400M for conversation; MiniLM + FAISS for CBT knowledge retrieval.
  4. Safety Layer — Built a 3-level crisis filter with keyword matching and clinical overrides; integrated 988/911 resource cards.
  5. Backend Routes — Developed REST endpoints for chat, mood history, sessions, and exercises; added WebSocket streaming and session-based rate limiting.
  6. Frontend UI — Built 4-tab SPA (Chat, Mood, Exercises, Profile) with dark-mode CSS design system, Chart.js dashboards, and breathing exercise animations.
  7. TTS / STT — Wired gTTS backend (5 Indian languages) with browser SpeechSynthesis fallback; Web Speech API for voice input.
  8. Testing & Bug Fixes — Ran live API tests; fixed 8+ bugs including FK constraints, session validation, crisis thresholds, and field mismatches.
  9. Packaging & Deployment — Wrote Dockerfile, docker-compose, Procfile, .env.example; pushed to GitHub.

Conclusion

MindEase PRO successfully demonstrates how transformer-based NLP, RAG pipelines, and real-time web technologies can be combined into a production-ready mental health support system. The application detects emotional state, retrieves evidence-based CBT strategies, and responds empathetically — all within a sub-second latency on consumer hardware. The inclusion of Indian language TTS (Kannada, Hindi, Tamil, Telugu, English) makes it accessible to a broader audience.

Future enhancements could include a fine-tuned mental-health-specific LLM, therapist escalation workflows, multi-user accounts, and mobile app packaging. The project is open-sourced on GitHub and containerised for one-command deployment.
